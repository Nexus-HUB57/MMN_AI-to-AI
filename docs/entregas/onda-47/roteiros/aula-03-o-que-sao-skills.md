# AULA 03 · O que são Skills de IA?
**Duração alvo:** 7-9 min · **Slug:** `curso-universo-ia-03`

## 🎯 Objetivo
O aluno entende que Skill = pacote reutilizável de capacidade IA (prompt + regras + memória + integrações) e sabe listar 5 Skills essenciais do MMN AI-to-AI.

## 🎬 Roteiro

### [00:00 – 00:30] Hook
> "Uma IA sem skill é como um funcionário sem manual. Uma IA COM skill é o funcionário mais produtivo que sua empresa já teve."

### [00:30 – 02:30] Definição
> "Skill é um bloco autônomo: **prompt** + **regras de negócio** + **memória** + **ferramentas conectadas**. É o que transforma 'peça pra IA' em 'a IA já sabe o que fazer'."

### [02:30 – 05:00] As 5 skills que todo afiliado Nexus precisa
1. **Skill de Vendas** — atende, qualifica, fecha.
2. **Skill de Recrutamento** — apresenta o plano, envia link, faz follow-up.
3. **Skill de Conteúdo** — cria post, roteiro, vídeo, ebook.
4. **Skill de Suporte** — tira dúvida, resolve reclamação.
5. **Skill de Análise** — lê seu dashboard e diz o próximo movimento.

### [05:00 – 07:00] Demo
Mostrar dentro da plataforma como ativar uma skill em 3 cliques.

### [07:00 – 08:30] Custo x retorno
> "Uma Skill bem construída paga o custo em 1 conversão. Depois, é lucro 24/7."

### [08:30 – 09:00] CTA
> "Aula 4: os TIPOS de agentes IA que existem e qual serve para cada situação."
