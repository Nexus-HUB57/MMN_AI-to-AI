# Guia de Uso - IA Content Hub Avançado

## Bem-vindo ao IA Content Hub!

O IA Content Hub é uma plataforma inteligente para gerar, otimizar e publicar conteúdo em múltiplas redes sociais usando inteligência artificial.

---

## 1. Começando

### 1.1 Criar Conta

1. Acesse https://ia-content-hub.example.com
2. Clique em "Criar Conta"
3. Escolha seu método de login:
   - Google
   - GitHub
   - Email e senha
4. Preencha seu perfil

### 1.2 Conectar Redes Sociais

1. Vá para **Configurações** → **Redes Sociais**
2. Clique em **Conectar** para cada rede desejada
3. Autorize o acesso
4. Pronto! Agora você pode publicar diretamente

**Redes suportadas:**
- Instagram
- Twitter
- LinkedIn
- TikTok

---

## 2. Gerar Conteúdo

### 2.1 Geração Básica

1. Clique em **Novo Conteúdo**
2. Preencha o prompt (o que deseja gerar)
3. Escolha o modelo de IA:
   - **Google Gemini**: Rápido e eficiente
   - **OpenAI GPT-4**: Mais criativo
   - **Modelo MMN**: Otimizado para redes sociais
4. Configure opções:
   - **Plataforma**: Instagram, Twitter, etc.
   - **Tom**: Profissional, Casual, Persuasivo, Humorístico
   - **Comprimento**: Número máximo de caracteres
   - **Hashtags**: Incluir automaticamente
   - **Emojis**: Incluir ou não
5. Clique em **Gerar**

### 2.2 Exemplo de Uso

```
Prompt: "Gere um post sobre os benefícios da inteligência artificial para pequenas empresas"

Configurações:
- Plataforma: LinkedIn
- Tom: Profissional
- Comprimento: 280 caracteres
- Hashtags: Sim
- Emojis: Não
```

**Resultado:**
> A IA está transformando a forma como pequenas empresas operam. Automação de processos, análise de dados em tempo real e atendimento ao cliente 24/7 são apenas algumas vantagens. Investir em IA agora é investir no futuro do seu negócio. 🚀 #IA #Negócios #Inovação

### 2.3 Gerar Variações

Quer múltiplas versões do mesmo conteúdo?

1. Clique em **Gerar Variações**
2. Escolha quantas variações deseja (2-5)
3. O sistema gera com diferentes tons:
   - Profissional
   - Casual
   - Persuasivo
   - Humorístico
4. Escolha a que mais gosta

## 3. Análise de Sentimento

### 3.1 Entender o Sentimento do Seu Conteúdo

Antes de publicar, analise o sentimento:

1. Gere ou cole o conteúdo
2. Clique em **Analisar Sentimento**
3. Veja:
   - **Score**: 0-100 (0 = muito negativo, 100 = muito positivo)
   - **Classificação**: Positivo, Neutro ou Negativo
   - **Palavras-chave**: Termos que indicam o sentimento
   - **Confiança**: Quão certo o modelo está

### 3.2 Exemplo

```
Conteúdo: "Adorei o novo produto! Superou minhas expectativas!"

Resultado:
- Score: 92
- Classificação: Positivo ✅
- Palavras-chave: adorei, superou, expectativas
- Confiança: 0.98
```

O sistema aprende com seu histórico e sugere:

2. **Melhor tom**: Qual tom gera mais engajamento

3. Clique em uma para gerar conteúdo baseado nela

## 5. Templates de Conteúdo

### 5.1 Usar Templates Pré-definidos

Templates economizam tempo:

1. Clique em **Templates**
2. Escolha uma categoria:
   - Promoção
   - Educação
   - Engajamento
   - Anúncio
   - Outro
3. Selecione um template
4. Preencha as variáveis
5. Customize conforme necessário

### 5.2 Criar Seu Próprio Template

1. Clique em **Novo Template**
2. Dê um nome e descrição
3. Escreva o conteúdo com variáveis:
   ```
   Olá {{nome}}, confira nosso novo {{produto}}!
   ```
4. Salve
5. Use sempre que precisar

## 6. Agendar Posts

### 6.1 Agendar para Depois

Não quer publicar agora?

1. Gere ou escolha conteúdo
2. Clique em **Agendar**
3. Escolha:
   - **Data e hora**
   - **Plataformas** (pode agendar para múltiplas)
   - **Mídia** (imagens/vídeos opcionais)
4. Clique em **Agendar**

O sistema publicará automaticamente no horário escolhido!

### 6.2 Gerenciar Posts Agendados

1. Vá para **Posts Agendados**
2. Veja lista de posts futuros
3. Edite ou cancele conforme necessário

## 7. Upload de Mídia

### 7.1 Adicionar Imagens e Vídeos

Suporte para:
- **Imagens**: JPG, PNG, WebP (máx 10MB)
- **Vídeos**: MP4, WebM (máx 100MB)

O sistema otimiza automaticamente:

- Redimensiona para cada plataforma

## 8. Analytics e Desempenho

### 8.1 Ver Estatísticas

1. Vá para **Analytics**
2. Escolha período:
   - Hoje
   - Esta semana
   - Este mês
3. Veja métricas:
   - **Visualizações**: Quantas pessoas viram
   - **Curtidas**: Quantas pessoas curtiu
   - **Compartilhamentos**: Quantas vezes foi compartilhado
   - **Comentários**: Quantos comentários recebeu
   - **Taxa de engajamento**: Porcentagem de interação

### 8.2 Comparar Desempenho

1. Selecione múltiplos posts
2. Clique em **Comparar**
3. Veja qual teve melhor performance
4. Use insights para futuras publicações

### 8.3 Exportar Relatórios

1. Vá para **Analytics**
2. Clique em **Exportar**
3. Escolha formato:
   - PDF
   - CSV
   - Excel

## 10. Dicas e Boas Práticas

### 10.1 Para Melhor Engajamento

1. **Use o tom certo**: Profissional para LinkedIn, casual para Instagram
2. **Poste no horário certo**: Use a recomendação de horário
3. **Inclua hashtags**: Aumenta alcance
4. **Use imagens**: Posts com imagem têm 2x mais engajamento
5. **Responda comentários**: Aumenta taxa de engajamento

### 10.2 Evite Erros Comuns

1. ❌ Postar conteúdo muito longo para Twitter
2. ❌ Usar tom profissional em TikTok
3. ❌ Postar no mesmo horário todos os dias
4. ❌ Ignorar comentários
5. ❌ Usar muitos hashtags (#)

### 10.3 Otimize Seu Perfil

1. Foto de perfil clara
2. Bio descritiva
3. Link para seu site
4. Posts consistentes

## 12. Troubleshooting

### Problema: "Erro ao conectar rede social"

**Solução:**
1. Verifique se a conta está ativa
2. Revogue acesso e reconecte
3. Tente em navegador diferente
4. Contate suporte

### Problema: "Post não foi publicado"

**Solução:**
1. Verifique conexão com internet
2. Veja logs em **Histórico**
3. Tente publicar novamente
4. Contate suporte se persistir

### Problema: "Geração de conteúdo muito lenta"

**Solução:**
1. Tente modelo diferente (mais rápido)
2. Reduza comprimento máximo
3. Tente em horário com menos uso
4. Upgrade para plano Pro
