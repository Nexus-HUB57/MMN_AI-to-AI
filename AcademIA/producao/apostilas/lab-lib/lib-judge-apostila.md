# 📘 Apostila · lib-judge-revisor

## 1. Resumo executivo

Documento canônico da Lib Nexus. Esta apostila padroniza o uso do conteúdo de referência `lib-judge-revisor` dentro da Academ'IA.

## 2. Conceitos-chave

| Conceito | Definição operacional |
|---|---|
| Documento canônico | fonte oficial de referência |
| Versionamento | SemVer aplicado a cada atualização |
| Escopo | limite do que o documento cobre |
| Audiência | níveis para os quais é obrigatório |

## 3. Passo a passo operacional

1. localizar o documento na Lib Nexus
2. ler do início ao fim sem pular seções
3. mapear o conteúdo na operação real
4. anotar dúvidas para discussão na comunidade
5. revisar o documento a cada release

## 4. Exemplo prático aplicado

O afiliado consulta `lib-judge-revisor` antes de tomar uma decisão operacional, evitando improviso e mantendo aderência aos padrões da plataforma.

## 5. Checklist de execução

- [ ] documento lido na íntegra
- [ ] aplicação mapeada
- [ ] dúvidas registradas
- [ ] revisão semestral agendada
- [ ] referência arquivada para consulta rápida

## 6. Erros comuns e mitigação

| Erro | Mitigação |
|---|---|
| Citar sem ler | exigir leitura integral |
| Ignorar versão | conferir SemVer atual |
| Aplicar fora do escopo | respeitar limite editorial |
| Não revisar após release | criar lembrete semestral |

## 7. Próximo passo

Manter a Lib Nexus como fonte oficial e revisar este documento sempre que houver release relevante da plataforma.
