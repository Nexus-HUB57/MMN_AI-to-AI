# 📘 Apostila · Primeiro Agente

## 1. Resumo executivo

Criar o primeiro agente operacional do afiliado com 4 skills base e 1 ciclo controlado. Esta apostila acompanha a aula de mesmo nome na trilha Agente da Academ'IA Nexus.

## 2. Conceitos-chave

| Conceito | Definição operacional |
|---|---|
| Agente | persona + bundle de skills + autonomia |
| Skill | unidade funcional executável (copy, segmentação, julgamento) |
| Autonomia | regra de autoaprovação e limites de risco |
| Judge | camada de revisão automática de outputs |
| Entitlement | permissão real do nível para executar a skill |

## 3. Passo a passo operacional

1. abrir o painel Admin Runtime
2. acionar a criação ou edição do agente
3. selecionar as skills compatíveis com o objetivo
4. definir autonomia conservadora no início
5. rodar um batch piloto pequeno
6. ler métricas e calibrar
7. escalar somente após estabilidade

## 4. Exemplo prático aplicado

Cenário: afiliado nível Operador com produto digital.

- selecionar segmento intenção morna
- gerar 3 variações de copy via copywriter-persuasivo
- validar via judge-revisor
- disparar para batch inicial de 30 contatos
- ler resposta em janela de 24h
- ajustar threshold se reprovação Judge passar de 10%

## 5. Checklist de execução

- [ ] objetivo do agente declarado em uma frase
- [ ] skills compatíveis com o objetivo
- [ ] autonomia inicial conservadora
- [ ] batch piloto pequeno definido
- [ ] critério de sucesso e parada definido
- [ ] janela de leitura agendada
- [ ] log do ciclo arquivado

## 6. Erros comuns e mitigação

| Erro | Causa | Mitigação |
|---|---|---|
| Reprovação Judge > 20% | copy genérica | refinar persona e exemplo |
| Latência alta | dispatcher frio | aquecer canal antes do batch |
| Fila travada | volume excessivo | dividir em batches menores |
| Métrica volátil | amostra pequena | rodar pelo menos 2 ciclos completos |

## 7. Próximo passo

Avançar para a próxima aula da trilha Agente conforme o catálogo oficial e cadastrar a apostila no painel AdminAcademia.
