# 📚 Apostilas Academ'IA — Gestão EAD

> **15 apostilas oficiais** para a formação completa de afiliados Nexus.
> Versão 2.0.0 · MMN AI-to-AI · 2026

Esta pasta contém as 15 apostilas oficiais do programa **Academ'IA · Gestão EAD** — o braço educacional da Nexus Affil'IA'te. As apostilas seguem o padrão MMN_IA (frontmatter, capa, sumário navegável, apêndices, rodapé padronizado) e são convertidas para PDF via Pandoc quando necessário.

A versão 2.0.0 (junho/2026) **expande** a versão 1.0.0 com **5 apostilas avançadas** (11-15) voltadas para operadores, arquitetos, white-labels e líderes de operação que precisam ir além do dia-a-dia do afiliado.

## 🔧 Como Compilar para PDF

```bash
# Requer pandoc + wkhtmltopdf
cd AcademIA/apostilas/
for f in *.md; do
  pandoc "$f" -o "../pdf/$(basename "$f" .md).pdf" \
    --pdf-engine=wkhtmltopdf \
    --css=../templates/pdf.css \
    --toc --toc-depth=2
done
```

## 🆕 O que mudou na v2.0

A v2.0 representa a **segunda onda** da Academ'IA — focada em quem já domina o básico e quer operar, arquitetar ou escalar.

| Adição | Foco |
|--------|------|
| Apostila 11 — SHO | Operadores de plataforma, on-call, SRE |
| Apostila 12 — IOAID | Arquitetos de solução, SRE, engenheiros sênior |
| Apostila 13 — Marketplace | Autores de skill, autores iniciantes |
| Apostila 14 — Multi-Tenant | White-labels, parceiros regionais |
| Apostila 15 — Métricas & ROI | Heads de operação, board, consultores |

Juntas, essas apostilas formam o **bloco avançado** necessário para a **Certificação CNX (Nexus Master)**, a nova trilha de certificação de elite da Academ'IA.
