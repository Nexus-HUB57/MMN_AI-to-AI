1. O que são LLMs?

Large Language Models são redes neurais treinadas com bilhões de palavras. Eles preveem a próxima palavra em uma sequência.

📊 Exemplo: GPT-3 foi treinado com 570 GB de textos.
[Ilustração: enorme biblioteca de dados alimentando um cérebro digital]

2. Arquitetura Transformer

Apresentada no paper "Attention is All You Need" (2017). O mecanismo de atenção permite que o modelo foque em partes relevantes do texto.

[Diagrama do Transformer: input embedding → atenção multi-head → feed forward → output]

4. Engenharia de Prompt

Técnicas para obter melhores respostas:

  • Zero-shot: sem exemplos
  • Few-shot: com 2-3 exemplos
  • Chain-of-Thought: "pense passo a passo"
[Comparação: prompt ruim vs. prompt otimizado]

5. Alucinações

Modelos de linguagem podem inventar fatos com confiança. Isso é chamado de alucinação.

🔍 Exemplo: ChatGPT pode criar referências bibliográficas falsas.

6. RAG (Retrieval-Augmented Generation)

RAG combina busca em uma base de dados + geração de texto, reduzindo alucinações e permitindo respostas atualizadas.

[Fluxo: Pergunta → busca vetorial → contexto + LLM → resposta]

7. Ética e plágio

LLMs podem reproduzir textos protegidos por direitos autorais. Além disso, perpetuam vieses dos dados de treino.

8. Agentes autônomos

Futuro: LLMs que controlam ferramentas (navegador, código, APIs) para executar tarefas complexas. Ex: AutoGPT, BabyAGI.

9. Glossário + Desafio

Token: unidade de texto
Context window: limite de tokens
Fine-tuning: ajuste fino

Desafio: Crie um prompt que force o ChatGPT a resolver um problema de lógica usando chain-of-thought.
