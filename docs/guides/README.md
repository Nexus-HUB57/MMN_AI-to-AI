## Uso dos Guias

### Para Administradores

Consulte `CONSOLIDATED_ADMIN_GUIDE.md` para:
- Gestión de usuarios y red
- Configuración de comisiones y pagos
- Monitoreo del sistema
- Gestión de marketplaces

### Para Afiliados

Consulte `CONSOLIDATED_AFFILIATE_GUIDE.md` para:
- Primeiros passos na plataforma
- Configuração e uso do Agente IA
- Gestão da rede e comissões
- Uso de materiais de marketing

### Para Desenvolvedores

Consulte `CONSOLIDATED_INTEGRATION_MANUAL.md` para:
- Integração via API tRPC
- Configuração de webhooks
- Conexão com marketplaces externos
- Autenticação e segurança
