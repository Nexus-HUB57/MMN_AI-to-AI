4 de 5 itens D18 implementados e validados em produção. **pgBouncer ficou como TODO**

## 1. Pivot exchange: Binance → Mercado Bitcoin ✅
**Bloqueio diagnosticado**: API Binance retorna `Service unavailable from a restricted location`
ao chamar do IP `143.95.213.237` (Brasil). As credenciais fornecidas não puderam ser usadas
diretamente; ficaram salvas em `.env` para futuro uso com proxy.

**Solução**: pivot para Mercado Bitcoin (BR nativo, sem geo-block).

- Adicionadas funções `fetchFromMercadoBitcoin()` e `fetchFromFoxbit()` em `backend/src/services/btc-quote-service.ts` (markers D18-mercadobitcoin, D18-orchestrator)
- Nova cadeia de fallback: **Mercado Bitcoin → Foxbit → Binance → CoinGecko**
- `bankingRouter.requestBtcWithdrawal` agora prioriza `MERCADO_BITCOIN_API_KEY` (marker D18-mb-withdraw)
- **Live validation**: `source:"mercadobitcoin"`, `custodian:"Mercado Bitcoin"`, BRL/BTC = **R$ 316.371** (preço real spot)
