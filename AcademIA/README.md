## 🚀 O que é a Academ'IA?

A **Academ'IA** é um programa educacional estruturado que leva um afiliado de **zero a Elite** em 90 dias, com conteúdo técnico-operacional sobre como construir, operar, e escalar um ecossistema de IA para marketing multinível.

Não é um curso genérico sobre "IA para negócios". É um **sistema de treinamento técnico-prático** desenhado especificamente para o operador do ecossistema **Nexus HUB57** — um ambiente distribuído com IOAID, SHO, agentes autônomos, Judge Revisor, e federação.

## 💻 Stack Tecnológico

**Backend / Infra:**
- IOAID: 5 módulos (autenticação, event bus, agent runtime, judge, monitoring)
- SHO: Self-Healing Orchestrator (uptime 99.97%, MTTR < 2min)
- PostgreSQL + Redis + Prometheus + Grafana
- Docker Compose para setup local

**Agentes:**
- LLM: OpenAI, Anthropic, ou modelo local fine-tuned
- 18 skills base (copywriter, segmenter, judge, etc.)
- Judge Revisor: 3 modos (Revisão, Bloqueio Parcial, Bloqueio Total)
- Guardrails + Logs + Fallback + Métricas

**UX / Frontend:**
- HTML5 + CSS3 (custom dark theme)
- JavaScript vanilla (sem dependências externas)
- 100% open source
- Acessível (modo claro/escuro, impressão otimizada)
