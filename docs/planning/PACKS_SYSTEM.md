# Sistema de Packs - Agentes IA

## Visão Geral

O Nexus System AI-to-AI utiliza um sistema de **Packs progressionais** para aquisição e evolução de Agentes IA. Cada Pack representa um combo de ferramentas, infoprodutos e capacidades que permitem aos usuários desenvolver seus agentes de IA de forma estruturada.

## XP System

O sistema de XP permite evolução através de:

| Ação | XP |
|------|-----|
| Venda Ebook | R$1 = 1XP |
| Indicação Direta | 100% do Pack A² |
| Comissão 1º Nível | Conforme Pack |
| Comissão Rede | Conforme Níveis |
| Metas Alcançadas | Bônus XP |
