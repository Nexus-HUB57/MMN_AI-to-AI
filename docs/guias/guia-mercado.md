## 2. Conectando um Marketplace

### Pré-requisitos

- Conta ativa no marketplace desejado
- Credenciais de acesso (e-mail/senha)
- Dados para pagamento configurados

### Processo de Conexão

1. Acesse **Marketplaces** no menu principal
2. Localize o marketplace desejado na lista
3. Clique em **"Conectar"**
4. Insira suas credenciais
5. Autorize a integração
6. Aguarde a aprovação (1-24 horas)

---

## 3. Gestão de Produtos

### Importando Produtos

**Método 1 - Importação Automática**
1. Acesse Marketplaces → Importar
2. Selecione categorias de interesse
3. Configure margem de lucro
4. Clique em "Importar Selecionados"

**Método 2 - Importação Manual**
1. Acesse Marketplaces → Produtos
2. Clique em "Adicionar Produto"
3. Insira URL do produto no marketplace
4. Configure preços e margem

### Sincronização de Preços

| Estratégia | Margem Recomendada |
|------------|-------------------|
| Agressiva | 5-10% |
| Moderada | 10-20% |
| Premium | 20-30% |
