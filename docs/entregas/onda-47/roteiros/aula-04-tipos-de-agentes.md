# AULA 04 · Tipos de Agentes IA e diferenças
**Duração alvo:** 7-8 min · **Slug:** `curso-universo-ia-04`

## 🎯 Objetivo
O aluno distingue 4 tipos de agentes e escolhe o adequado para cada caso do seu negócio.

## 🎬 Roteiro

### [00:00 – 00:30] Hook
> "Nem toda IA é igual. Escolher o agente errado é como usar uma Ferrari para carregar tijolo."

### [00:30 – 05:30] Os 4 tipos
1. **Reativo (simples):** responde a estímulos, sem memória. Ex: FAQ automático.
2. **Deliberativo:** planeja passos antes de agir. Ex: montar uma campanha.
3. **Aprendente (RL):** melhora com cada tentativa. Ex: ajuste de preço em marketplace.
4. **Híbrido (agentic):** combina tudo, usa múltiplas skills. Ex: o Agente MMN AI-to-AI.

### [05:30 – 07:00] Matriz de decisão
| Necessidade | Tipo |
|---|---|
| Resposta imediata simples | Reativo |
| Estratégia | Deliberativo |
| Melhorar sozinho | Aprendente |
| Fazer tudo | Híbrido |
