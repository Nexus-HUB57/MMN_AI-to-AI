## 🎨 Identidade Visual Padrão

### Paleta Cyberpunk
- **Preto profundo:** `#0A0E27` (background)
- **Rosa neon:** `#FF00FF` (destaque primário)
- **Ciano elétrico:** `#00FFFF` (destaque secundário)
- **Branco puro:** `#FFFFFF` (texto)
- **Vinho escuro:** `#722F37` (accent Ive)
- **Azul aço:** `#4682B4` (accent Alencar)
- **Dourado Nexus:** `#FBBF24` (highlights)

### Tipografia
- **Títulos:** Sans-serif geométrica bold (Orbitron / Audiowide)
- **Corpo:** Sans-serif clean (Inter / Roboto)
- **Código:** Monospace (Fira Code / JetBrains Mono)

### Aspect Ratio
- **Padrão:** 16:9 (1920x1080)
- **Vertical:** 9:16 (1080x1920) — para mobile
- **Quadrado:** 1:1 (1080x1080) — para thumbnails sociais
