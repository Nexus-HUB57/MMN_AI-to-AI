Infraestrutura
🏗️ IOAID

Infraestrutura Operacional de Inteligência Distribuída. Sistema que processa requisições em 5 módulos (autenticação, event bus, agent runtime, judge, monitoring) em < 800ms.

→ Curso 01-Entendendo-IOAID
Infraestrutura
🛡️ SHO

Self-Healing Orchestrator. Sistema imunológico que monitora, diagnostica, cura e aprende com incidentes. Meta: 99.97% uptime, MTTR < 2 min.

→ Curso 02-Sistema-SHO
Infraestrutura
📨 Event Bus

Sistema de mensageria assíncrona (geralmente Redis Streams) que roteia eventos entre módulos do IOAID. Garante que nenhum evento se perde.

Infraestrutura
🧠 Agent Runtime

Módulo do IOAID onde o LLM é executado com system prompt, contexto, e ferramentas. É o "cérebro" de cada agente.

Infraestrutura
⏱️ MTTR

Mean Time To Recover. Tempo médio para recuperar de uma falha. Meta do SHO: < 2 minutos para 90% dos incidentes.

Agentes
🤖 Agente

Sistema que combina LLM + identidade + ferramentas + memória para executar tarefas de forma autônoma. Não é um chatbot — é um operador digital.

→ Curso 00-Primeiro-Agente
Agentes
🛠️ Skill

Capacidade específica de um agente. Ex: copywriter (escreve mensagens), audience-segmenter (segmenta público), judge (avalia compliance).

→ Curso 01-Skills-Essenciais
Agentes
⚖️ Judge Revisor

Filtro ético e de compliance que valida toda resposta antes de enviar. 3 modos: Revisão, Bloqueio Parcial, Bloqueio Total.

→ Curso 03-Judge-Revisor
Agentes
🎭 System Prompt

Instrução que define identidade, tom, regras, e limites de um agente. Mais impactante que o modelo escolhido — é o que diferencia 2 agentes com mesmo LLM.

Agentes
🚧 Guardrail

Limite explícito do que um agente pode/não pode fazer. Implementado via Judge + system prompt + validação pós-output.

Marketing
💰 CAC

Customer Acquisition Cost. Custo total para adquirir 1 cliente. Inclui: anúncios, ferramentas, tempo da equipe, infraestrutura.

→ Curso 00-Otimização-Conversão
Marketing
💎 LTV

Lifetime Value. Receita total esperada de um cliente ao longo do relacionamento. Meta: LTV/CAC > 3.

Marketing
📊 Coorte

Grupo de usuários que compartilham uma característica temporal (ex: "todos que entraram em Janeiro"). Análise de coorte = rastrear comportamento por grupo.

→ Curso 03-Coortes-Churn
Marketing
📉 Churn

Taxa de cancelamento. % de clientes que saem em um período. Reduzir churn aumenta LTV sem aumentar CAC.

Marketing
🔬 A/B Testing

Teste comparativo entre 2 variantes (A vs B). Muda 1 coisa por vez. Precisa de poder estatístico (n mínimo) para conclusões válidas.

→ Curso 02-AB-Test-Judge
Negócio
🏢 Multi-tenant

Arquitetura onde 1 sistema serve múltiplos clientes (tenants) com isolamento de dados. White-label = o cliente vende como se fosse seu produto.

→ Curso 01-Multi-tenant
Negócio
🌐 Federação

Múltiplos agentes cooperando, com mTLS e níveis de confiança. Permite escalar sem ponto único de falha.

→ Curso 02-Federação
Negócio
📐 Blueprint Elite

Padrão operacional dos top 10% afiliados Nexus. 5 pilares: técnico, escala, governança, relacionamento, visão.

→ Curso 00-Blueprints-Elite
Segurança
🔐 mTLS

Mutual Transport Layer Security. Autenticação criptográfica mútua — ambos os lados precisam de certificado. Obrigatório em federação.

Segurança
🛡️ LGPD

Lei Geral de Proteção de Dados. Regula coleta, armazenamento, e uso de dados pessoais no Brasil. Respeitar opt-out é obrigação legal.
