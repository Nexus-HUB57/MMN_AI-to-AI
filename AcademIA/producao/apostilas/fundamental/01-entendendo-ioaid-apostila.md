# 📘 Apostila · 01 Entendendo o IOAID

## 1. Resumo executivo

O IOAID é a Infraestrutura Operacional de Inteligência Distribuída do Nexus. Esta apostila explica suas 5 camadas, mostra o fluxo típico do afiliado e prepara o aluno para entender o SHO na aula seguinte.

## 2. Conceitos-chave

| Camada | Função |
|---|---|
| Dados | entrada e contexto |
| Agentes | inteligência aplicada |
| Orquestração | distribuição de tarefas |
| Validação (Judge) | aprova, revisa, bloqueia |
| Execução | dispara em canais reais |

## 3. Passo a passo operacional

1. identificar a camada em que sua dúvida está
2. seguir o fluxo padrão: dado → agente → orquestração → judge → execução
3. validar resultado pelo Judge antes da execução
4. arquivar evidência do ciclo
5. iterar a partir do resultado

## 4. Exemplo prático aplicado

Fluxo típico: lead chega → segmentação aplica intenção → copy é gerada → Judge revisa → disparo no WhatsApp → leitura de resposta em janela de 24h.

## 5. Checklist de execução

- [ ] sei explicar IOAID com minhas palavras
- [ ] sei nomear as 5 camadas
- [ ] consigo mapear um fluxo simples
- [ ] entendo onde o Judge atua
- [ ] tenho um exemplo aplicado na minha operação

## 6. Erros comuns e mitigação

| Erro | Mitigação |
|---|---|
| confundir camadas | usar diagrama oficial |
| pular o Judge | tornar revisão obrigatória |
| operar sem evidência | exigir log do ciclo |
| achar IOAID complexo demais | revisar a aula 00 antes |
